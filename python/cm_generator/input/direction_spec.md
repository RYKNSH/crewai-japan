## 📐 アニメーションスタイル

### 基本方針
| 項目 | 仕様 |
|------|------|
| **スタイル** | モダンカートゥーン（Rubber Hose + Squash & Stretch） |
| **フレームレート** | 24fps（動きの激しいシーンは伸縮を強調） |
| **参考作品** | Cuphead, Hilda, Adventure Time, Gravity Falls |

### アニメーション原則
```
1. Squash & Stretch - 小人の動きは常に弾むように
2. Anticipation - 大きなアクション前に予備動作
3. Follow Through - 動きの後に慣性で揺れる
4. Exaggeration - 感情表現は大げさに
```

## ✨ エフェクト詳細

### Scene 2: 願いの瞬間
```
[エネルギー波]
- 主人公の口から同心円状に広がる
- 色: 白→虹色グラデーション
- 速度: 0.5秒で画面外へ
```

### Scene 3: エージェント誕生
```
[パーティクル集合]
- 光の粒子が渦を巻いて集まる
- 集合 → 閃光 → 小人登場
- 各小人の色に対応したパーティクル

[POP エフェクト]
- スモーク（白）+ 星型キラキラ
- 「ボン！」という膨張と収縮
```

### Scene 6: 作業シーン
```
[スピードライン]
- 背景に放射状の速度線
- 色: 淡いグレー（邪魔にならない程度）

[モーションブラー]
- 手の動きにブラー適用
- フレーム数を意図的に減らし「残像」表現
```

### Scene 7: 完成
```
[達成エフェクト]
- 紙吹雪（多色）
- 花火のような放射
- グロー効果（成果物が光る）
```

## 💬 テキストアニメーション

### タイポグラフィ
| 用途 | フォント参考 | スタイル |
|------|------------|---------|
| セリフ | 丸ゴシック系 | 手書き風、ポップ |
| テロップ | モダンサンセリフ | クリーン、読みやすい |
| ロゴ | カスタム | テック感 + 親しみやすさ |

### アニメーション手法
```
[セリフ]
- Bounce In: 文字がバウンドしながら登場
- Scale: 強調部分を一瞬大きく

[テロップ]
- Fade + Slide: フェードインしながらスライド
- Typewriter: タイピング風に1文字ずつ

[ロゴ]
- 光の軌跡を残しながらフェードイン
- 最後に小人たちがロゴの周りに集合
```

## 🎭 キャラクター演技指示

### 主人公
| シーン | 表情 | ボディランゲージ |
|--------|------|----------------|
| 1 | 😫 困り眉、下がった口角 | 猫背、頭を抱える |
| 2 | 😆 大きく開いた口、輝く目 | 両手を上げる、背筋伸ばす |
| 3 | 😲 驚きで目が大きく | 後ろに仰け反る |
| 8 | 😍 ハート目、大きな笑顔 | ガッツポーズ、跳ねる |

### 小人エージェント共通
```
[基本動作]
- 常に弾むような歩き方（ホップステップ）
- 頭身は2-2.5頭身（かわいさ重視）
- 目で感情表現（大きな目のアドバンテージ活用）

[作業中]
- 真剣な表情でも可愛らしく
- 時々見せる「できた！」のガッツポーズ
- 仲間との小さなハイタッチ
```

## 📌 補足資料

### 参考アニメーション
1. **Cuphead** - ゴム質の滑らかなアニメーション
2. **Gravity Falls** - ポップなキャラデザと表情
3. **Hilda** - 温かみのある色使い
4. **Adventure Time** - シンプルで効果的な表現

### 禁止事項
- ❌ リアルすぎる影や質感
- ❌ 暗すぎるカラーパレット
- ❌ 動きのない長い静止
- ❌ 複雑すぎるキャラクターデザイン
