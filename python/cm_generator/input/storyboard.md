# 🎬 小人エージェントCM ストーリーボード

> **コンセプト**: 「願いを叫ぶと、小人エージェントたちが飛び出して仕事をこなす」カートゥーンアニメーションCM

---

## 📋 基本情報

| 項目 | 内容 |
|------|------|
| **タイトル** | 「CrewAI - あなたの願いを叶える小さな仲間たち」 |
| **尺** | 30秒～45秒 |
| **スタイル** | カートゥーン / ポップ / エネルギッシュ |
| **ターゲット** | クリエイター、ビジネスパーソン、テック好きユーザー |

---

## 🎞️ シーン構成

### Scene 1: オープニング（0:00～0:03）
**「主人公の日常」**

```
┌────────────────────────────────────┐
│                                    │
│    😫 ← 主人公（デスクで頭を抱える）  │
│   ┌──────┐                         │
│   │ 📋📋📋 │  ← 山積みのタスク        │
│   └──────┘                         │
│                                    │
└────────────────────────────────────┘
```

**アクション**:
- 主人公がデスクで山積みのタスクを前に困り顔
- 書類やタスクリストが画面を埋め尽くす
- BGM: 重苦しい音楽（すぐに転換）

**セリフ/テキスト**:
> 「こんなに仕事があるのに、一人じゃ無理だよ...」

---

### Scene 2: 願いの瞬間（0:03～0:07）
**「叫びと変化」**

```
┌────────────────────────────────────┐
│          ✨💫✨                     │
│                                    │
│    😆 ← 主人公（両手を上げて叫ぶ）   │
│   「誰か手伝って〜！」              │
│          ⚡                        │
│                                    │
└────────────────────────────────────┘
```

**アクション**:
- 主人公が立ち上がって空に向かって叫ぶ
- 叫びに合わせてエネルギー波が広がる
- 画面がキラキラと輝き始める
- BGM: 劇的な転換、アップテンポに変化

**セリフ/テキスト**:
> 「だれかー！このレポート作って、データまとめて、プレゼン資料も作ってほしい〜！」

---

### Scene 3: エージェント誕生（0:07～0:12）
**「小人たちの飛び出し」**

```
┌────────────────────────────────────┐
│    ✨ POP! ✨  POP! ✨  POP! ✨     │
│                                    │
│   🧑‍💻  📊  ✍️  🔍  📱            │
│    ↓    ↓    ↓    ↓    ↓          │
│   それぞれ個性的な小人エージェント   │
│                                    │
│    😲 ← 驚く主人公                  │
└────────────────────────────────────┘
```

**アクション**:
- 主人公の頭上から光の粒子が集まる
- 「ポン！ポン！ポン！」と次々に小人エージェントが生成
- 各エージェントは異なる色・特徴を持つ
  - **青い帽子**: リサーチャー（虫眼鏡を持つ）
  - **緑のエプロン**: ライター（ペンを持つ）
  - **オレンジのゴーグル**: データアナリスト（グラフを持つ）
  - **紫のベレー帽**: デザイナー（パレットを持つ）
  - **赤いネクタイ**: マネージャー（バインダーを持つ）

**SE（効果音）**:
- 「ポヨン！」「シャキーン！」などポップな登場音

---

### Scene 4: タスク読み込み（0:12～0:17）
**「情報収集フェーズ」**

```
┌────────────────────────────────────┐
│                                    │
│  📚🔍     📋✏️     📊👀            │
│  小人A     小人B     小人C          │
│   │        │        │              │
│   └────────┼────────┘              │
│            ↓                       │
│      💭💭💭 (アイデアが浮かぶ)       │
└────────────────────────────────────┘
```

**アクション**:
- 各小人がタスクの紙や本を手に取る
- 紙が小人の目の前でバッと広がる
- 内容を読むとき、目がキラキラ光る
- 読み込んだ情報が頭上に吹き出しで表示

**演出ポイント**:
- スピード感のあるカット割り
- 紙がめくれる音、タイピング音のレイヤー
- 各小人の表情が「なるほど！」と理解する瞬間

## 🎨 キャラクターデザイン参考

### 小人エージェントの特徴

| エージェント | 色 | アクセサリー | 性格 |
|------------|-----|------------|------|
| リサーチャー | 青 | 虫眼鏡、本 | 知的、好奇心旺盛 |
| ライター | 緑 | ペン、ノート | クリエイティブ、情熱的 |
| アナリスト | オレンジ | ゴーグル、グラフ | 論理的、正確 |
| デザイナー | 紫 | ベレー帽、パレット | 芸術的、繊細 |
| マネージャー | 赤 | ネクタイ、バインダー | リーダーシップ、統率力 |

### デザインポイント
- 身長: 主人公の手のひらサイズ
- 体型: ずんぐりむっくり（かわいさ重視）
- 表情: 大きな目でリッチなエクスプレッション
- 動き: バネのような弾むアニメーション

## 📝 制作メモ

### トーン＆マナー
- **明るく楽しい**: 重い雰囲気は最初の3秒だけ
- **スピード感**: テンポよく、視聴者を飽きさせない
- **共感性**: 「あるある」感で視聴者の心をつかむ
- **驚き**: エージェント誕生シーンでインパクト

### 避けるべき表現
- ❌ 暗すぎる・重すぎる雰囲気
- ❌ 複雑すぎる説明
- ❌ 静止画が長すぎる
- ❌ キャラクターの表情が乏しい
